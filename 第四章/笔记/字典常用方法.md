``````python
""" 字典方法
    # clear方法 #

    D.clear() -> None
    
    简单描述：清空D中所有的项(items)
    返回值： None

    用于清空字典中的键，但并不清除值（自己的理解）
    除非清除该键以后没有其他标识符再次引用这个值


    # copy方法 #

    D.copy() -> dict

    简单描述：获取D的复制对象
    返回值：D的(浅)复制对象

    用于复制字典中的项（items/键值对）
    注意这里是浅复制，复制以后的键值对共享未修改的内存
    深复制请使用copy模块内的deepcoy方法


    # fromkeys方法 #

    D.fromkeys(iterable, value=None)

    通过迭代器中的元素作为键，可选的参数value将作为
    每个元素的默认值，可以自行修改value的值

    # get方法 #
    D.get(k[, d]) -> D[k] or d, d default to None
    
    简单描述： 尝试获取D中k键对应的值
    返回值：如果不存在则返回d，d默认为None
    
    
    # items方法 #

    D.items() -> a set-like object providing a view on D's items

    简单描述：获取字典所有的项(键值对)
    返回值：返回类set的类型（不会有重复元素），其中每个元素里
    的子元素的第一个元素表示键，第二个表示值


    # keys方法 #

    D.keys() -> a set-like object providing a view on D's keys

    简单描述：获取字典所有的键
    返回值：返回类set的类型（不会有重复元素），每个元素表示一个键


    # pop方法 #

    D.pop(k[, d]) -> remove key, and return corresponding value

    简单描述： 获取指定键的相关值
    返回值： 返回参数键对应的值，如果没有则返回d。d未定义则抛出ValueError


    # popitem方法 #

     D.popitem() -> (k, v)

    简单描述：随机弹出一个项（因为字典是无序的）
    返回值： 项（键值对）


    # setdefault方法 #

    D.setdefault(k[, d])  -> D.get(k,d), also set D[k]=d if k not in D

    简单描述： 尝试获取参数键的对应值，如果未定义则设置为默认值
    返回值： k的对应值，或者默认值d，如果d未定义则默认为None


    # update方法 #

    D.update -> None.

    简单描述：从其他字典中更新当前字典中的元素（不存在则添加）
    返回值: None


    # values方法 #

    D.values() -> an object providing a view on D's values

    简单描述：获取字典中所有的值
    返回值：字典中所有值的对象（如迭代器，序列）
"""


# 浅复制

l1 = [1, 2, 3, [5,4,6] ]
l2 = l1.copy()

# l1 R => [1, 2, 3, [5,4,6] ]
# l2 R => [1, 2, 3, [5,4,6] ]

# 我们只想给l1进行排序
l1[3].sort()

# 但是l2的值也跟着改变了
# l1 R => [1, 2, 3, [4, 5, 6]]
# l2 R => [1, 2, 3, [4, 5, 6]]


# 深度复制

import copy

l1 = [1, 2, 3, [5,4,6] ]
l2 = copy.deepcopy(l1)

# l1 R => [1, 2, 3, [5,4,6] ]
# l2 R => [1, 2, 3, [5,4,6] ]

l1[3].sort()

# 达到了我们想要的目的
# l1 R => [1, 2, 3, [4, 5, 6]]
# l2 R => [1, 2, 3, [5, 4, 6]]


``````
